### 2. Назначение и цели создания системы

**Вид автоматизируемой деятельности:**  
Учет и управление деятельностью гаражного хозяйства.

**Перечень автоматизируемых процессов:**
- учет транспортных средств;
- учет технического состояния автомобилей;
- учет ремонтов и технического обслуживания;
- хранение данных о водителях и ответственном персонале;
- формирование отчетов.

**Показатели, достигаемые в результате внедрения системы:**
- повышение точности учета информации;
- сокращение времени обработки данных;
- снижение количества ошибок при ведении документации;
- повышение эффективности контроля автопарка.

**Требования к системе в целом:**

- защита информации от несанкционированного доступа.

**Требования к программному обеспечению ИС:**

- использование системы управления базами данных;
- возможность резервного копирования данных.

**Требования к техническому обеспечению АС:**
- персональный компьютер;
- устройства ввода и вывода информации;
- при необходимости — доступ к сети Интернет.
